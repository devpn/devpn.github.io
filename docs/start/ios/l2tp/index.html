L2TP for iOS 使用教程

注意

L2TP 连接协议已经被国家防火墙 GFW 精准识别并阻断，不建议使用。

备注

下面使用 iPhone 来演示如何配置，iPad 的配置方法也是类似的，网上有相关教程。

点击 设置 ，进入系统的设置，点击 通用

点击 VPN ，进入 VPN 设置

点击 添加 VPN 配置...

协议类型选择 L2TP ； 描述 栏中填入 devpn-l2tp ； 服务器 栏中填入服务器域名，点击下面按钮查看服务节点列表； 帐户 栏中填入您的用户名； 密码 栏中填入您的连接密码； 密钥 栏中填入devpn；然后点右上角的"存储"按钮

VPN 配置中会看到刚设置的"devpn-l2tp"，选中后打开上方的 VPN 连接

设置 DNS，防止 DNS 污染（不设置可能会出现连接成功但国外网站还是上不了的情况）​

点击设置中的 Wi-Fi

点击当前使用中的网络

设置 DNS 为 8.8.8.8, 8.8.4.4

注：任何 iOS 设备在锁屏后，VPN 连接会自动断开，属于正常现象。
