L2TP for Mac OS 使用教程

注意

L2TP 连接协议已经被国家防火墙 GFW 精准识别并阻断，不建议使用。

点击屏幕下方的 设置图标 ，进入控制面板的设置，单击 网络 图标，进入网络的设置

单击左下方的 ＋ 号， 接口 栏中选择 VPN ，选择后，下面的选项随之变化； VPN 类型 栏中选择 PIPSec 上的 L2TP ；** 服务名称 **中填入 devpn-l2tp ；最后点击 创建 按钮

服务器地址 中填入服务器域名，点击下面按钮查看服务节点列表； 帐户名称 中填入您的用户名；勾选 在菜单栏中显示 VPN 状态 ，然后点击 鉴定设置... 按钮

密码 中填入您的连接密码，共享的密钥 中填入 devpn ，然后点击 好 按钮

点击 高级... 按钮，选中 通过 VPN 连接发送所有流量 ，单击上方 DNS 标签

添加 8.8.8.8 和 8.8.4.4 到 DNS 服务器中，最后点击 好 按钮

点击 应用 按钮，顶部菜单栏中显示连接 VPN 图标，方便快速连接或断开 VPN；点击 连接 选项，连接 VPN
