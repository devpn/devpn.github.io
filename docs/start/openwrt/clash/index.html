Clash for OpenWRT 使用教程

欢迎使用

Clash 具有强大的路由规则，只需要一个 URL 链接就可以轻松配置成功，支持 Reality 和 Hysteria 等多种连接协议，速度和稳定性都比较优秀，推荐优先选择。

备注

OpenWRT 有很多发行版本，支持的硬件设备非常多。这里以 iStoreOS 为例，安装 OpenClash 实现科学上网。

iStoreOS 固件下载地址：https://fw.koolcenter.com/iStoreOS/

下载安装​

点击对应架构的软件下载：

  • aarch64 平台
  • x64 平台

适用于 aarch64_cortex-a53 平台的机器 (EasePi ARS2、小米AX3600、AX9000、红米AX6等)。

也兼容 aarch64_generic 平台，例如 R2S、R4S、R5S、R68S 等。

适用于 KoolCenter iStoreOS 22.03.X 的软件包。

适用于 x86_64 平台的机器。

适用于 KoolCenter iStoreOS 22.03.X 的软件包。

下载后，来到 iStore 应用商店页面，点击手动安装，然后点击选择上传或者直接拖放文件。

注意需要路由器自身联网正常，部分依赖依然要通过 opkg 软件源在线安装。如果安装失败，检查路由器自身的网络情况，特别是旁路由模式下最容易出现网络问题。

插件设置​

安装完成后，来到 服务 > OpenClash > 插件设置。

在 模式设置 选项卡下，选择 使用 Meta 内核，运行模式为 Redir-Host，网络栈类型为 Mixed.

添加订阅​

请先复制订阅链接。如果导入失败，请使用备用订阅链接代替。

在 配置订阅 选项卡下。

先启用 自动更新，再点击 新增 按钮。

粘贴订阅地址，修改 User-Agent 为 clash.meta，完成后点击下方的 保存配置 按钮。

回到配置订阅界面，点击下方的 保存配置，然后点击 更新配置 按钮。

开启代理​

来到 运行状态 选项卡下。

等待程序启动运行，出现图上的状态表示已经正常运行。 点击任意一个 控制面板 按钮，可以查看节点列表，如果节点不稳定，请选择其他的节点。

配置完成，享受自由的互联网吧。
