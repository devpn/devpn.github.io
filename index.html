哪种连接协议最适合我

目前最好用的连接协议是 Trojan，使用 Clash 客户端进行连接。 时代在变，同时国家防火墙 GFW 也在不断地进步，有些连接使用一段时间就会被识别并阻拦，故每隔一段需要调整连接协议以获得稳定的连接。 并非每个 VPN 协议都适用于所有操作系统，确保选择适用于您要使用的设备的 VPN 协议。 我们的 VPN 协议产品 PPTP PPTP（点对点隧道协议）是可用的最古老的 VPN 协议之一。 PPTP 速度极快，但这是由于使用的加密较弱。 它不像大多数安全 VPN 协议那样使用 AES，而是使用 Microsoft 的点对点加密，该加密存在许多安全漏洞。 因此，PPTP 已经破解了十多年，MacOS 也不再支持 PPTP 协议。 对于那些不介意安全并且只想解锁网站的人来说，PPTP 是一个绝佳的选择。 这意味着您的互联网速度不会受到太大影响，从而在视频流式传输时获得良好的体验。 如果您完全关心您的安全性，您应该完全避免 PPTP 并使用不同的 VPN 协议。 L2TP/IPSec L2TP/IPSec 是目前可用的另一种最广泛支持的 VPN 协议。 它在包括 Windows 和 Android 在内的许多操作系统中具有原生支持。 L2TP/IPSec 的工作方式与 PPTP 类似，因为它使用 MPPE，但它在安全的 AES 隧道内分层。 这只会增加更多的开销，并且添加 MPPE 并没有好处，因为它已经被破解了。 L2TP/IPSec 最多支持 AES-256 位加密。 让 IPSec 通过防火墙也可能有点痛苦。 L2TP/IPSec 是一个可靠的选择，尤其是对于 Android 用户，因此最好将其作为备份选项。...

Clash for Windows 使用教程

下载安装 请点击此处下载 Clash for Windows 安装文件 下载后运行 exe 文件以安装，可能会提示不安全，请忽略此提示。 由于 Clash for Windows 没有任何有效的数字签名，因此 SmartScreen 可能会弹出提示，请点击“更多信息”，然后选择“仍要运行”。 添加订阅 请先复制 Clash 配置链接，并将用户名修改为您在网站上注册的用户名，密码为注册时使用的密码。 https://goen.win/rule/用户名/密码 打开 Clash for Windows 应用程序，在左侧的标签页中选择 Profiles ， 在顶部输入您的 Clash 订阅链接，然后点击 Download 按钮。 注意：用户名和密码要填写您在网站上注册的用户名和密码 Clash for Windows 会自动拉取配置文件进行更新，如果一切顺利，你应当可以看到绿色提示信息「Success!」，并且可以看到一个新增的配置文件： 点击新增的配置文件来切换到该配置，然后点击「Proxies」标签页来切换接入点，将顶部的出站模式选择为「Rule」。 此模式下你的网络访问请求将通过 Clash for Winows 进行分流处理。 在「Proxy」策略组中选择所想要使用的接入点。Proxy 策略组是用于访问国际网络的默认策略，在不进行其他修改的情况下，所有国际网络的访问都通过 Proxy 策略组中选择的接入点进行。 返回到「General」部分，将「System Proxy」的开关更改为绿色或「On」状态即可开始使用。此外，如果您需要让 Clash for Windows 开机自启，可以将「Start with Windows」也更改为绿色或「On」。 配置完成，打开浏览器享受自由吧。

Clash for Android 使用教程

下载安装 请点击这里下载 Clash for Android 软件 下载后进行安装并打开。 添加订阅 请先复制 Clash 配置链接，并将用户名修改为您在网站上注册的用户名，密码为注册时使用的密码。 https://goen.win/rule/用户名/密码 在 Clash for Android 主页，依次点击 配置 > 新配置 > URL 。 在 URL 一栏中填入 Clash 配置订阅链接；自动更新 (分钟) 推荐填写 1440 ，即每 24 小时自动从链接中更新配置文件。 注意：用户名和密码要填写您在网站上注册的用户名和密码 完成后点击右上角的保存按钮，Clash for Android 将下载配置文件，请点击选中添加的配置文件。 开启代理 回到 Clash for Android 主页，启动服务。在弹出的设置 VPN 中点击 “允许”，Clash for Android 便开始接管系统流量。 点开 “代理”，可以对策略组进行选择。 配置完成，打开浏览器享受自由吧。

Clash for MacOS 使用教程

下载安装 请点击此处下载 ClashX 软件包 下载后运行 dmg 文件以安装。 第一次安装如果提示隐私安全的提示，请选择——系统偏好设置——隐私与安全，点击允许。 添加订阅 请先复制 Clash 配置链接，并将用户名修改为您在网站上注册的用户名，密码为注册时使用的密码。 https://goen.win/rule/用户名/密码 打开 ClashX，点击菜单栏中的 ClashX 图标 打开菜单，进入 配置 > 托管配置 > 管理。 在 托管的配置文件 窗口，点击 “添加” 在 Url 一栏中输入 Clash 配置订阅链接，Config Name 为配置文件名，可以随意填写或留空。 注意：用户名和密码要填写您在网站上注册的用户名和密码 添加成功后 ClashX 会自动从链接下载配置文件，等待 “更新时间” 刷新后，即可关闭窗口。 设置为系统代理 打开 ClashX 菜单中的 “设置为系统代理”，ClashX 便开始接管系统流量。 配置完成，打开浏览器享受自由吧。
